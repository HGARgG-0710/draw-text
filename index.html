draw-text

What is this?

Draw-Text is a simple webpage, that allows one to draw polygons with the usage of a basic coordinate-setting format, then export the results to png/jpg or any other desired image format.

Why?

Why not? It's fun, try!

Vocabulary

The language of the page contains five elementary commands:

  1. contour
  2. fill
  3. clear
  4. erase
  5. background

Besides background (which is just an alias for a case of 'fill'), they are all independent. Usage looks something like the following:

  1. contour (x1, y1)...(xn, yn) contour (x1, y1, c1)...(xn, yn, cn) contour (x1, y1, c1)->...->(xn, yn, cn)-> contour (x1, y1, c1)->(cc1)...->(ccn)(xn, yn, cn)->
  2. fill (x1, y1)...(xn, yn) fill (x1, y1, c1)...(xn, yn, cn)
  3. clear (x1, y1)...(xn, yn) clear (x1, y1)->...->(xn, yn)->
  4. erase (x1, y1)...(xn, yn)
  5. background c

Here, the x and y are the point coordinates, while -> are the connections between them (to connect i'th coordinate with the next one, write an 'arrow' between them...). The c, c1...cn are the colours' values. These are (currently) settable with hex as RGB color spectre values (note: in future, more color schemes are intended for support).
