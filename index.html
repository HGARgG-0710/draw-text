draw-this

What is this?

Draw-Text is a simple webpage, that allows one to draw polygons with the usage of simple coordinate-setting format, then export the results to png/jpg any other desired image format.

Why?

Why not? Try it!

Vocabulary

The language of the page contains five elementary commands:

  1. contour
  2. fill
  3. clean
  4. erase
  5. background

Besides background (which is just an alias for a case of 'fill'), they are all independent. Usage looks something like the following:

  1. contour (x1, y1, c1)...(xn, yn, cn) contour (x1, y1, c1)->...->(xn, yn, cn)->
  2. fill (x1, y1, c1)...(xn, yn, cn)
  3. clean (x1, y1, c1)...(xn, yn, cn) clean (x1, y1, c1)->...->(xn, yn, cn)->
  4. erase (x1, y1, c1)...(xn, yn, cn)
  5. background c
